# Numerical Analysis Suite
A Python collection focused on the intersection of calculus and computation.

## Core Modules

  - **Approximation**: Symbolic generation of Taylor & Fourier Series via SymPy with error-convergence plots in Matplotlib.
  - **Integration**: Algorithmic implementation of Simpson’s Rule and the Midpoint Method for precise numerical estimation.
  - **Analysis**: Automated limit evaluation and discontinuity detection for complex functions.
